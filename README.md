# SaferWebAPI

## Installation

Install using pip

```shell
pip install fastapi
```

```shell
pip install "uvicorn[standard]"
```

## Dependencies

* [python-safer](https://github.com/arthurtyukayev/python-safer)

Install python-safer lib using pip
```shell
pip install python-safer
```

## Run

Run the server with:

```shell
python -m uvicorn main:app --reload
```
